# AI-Based Loan Risk Prediction System

## Overview
This project is a full-stack web application that predicts the risk level of loan applicants using machine learning techniques.

## Key Features
- Loan application and approval system
- AI-based risk prediction using classification model
- Real-time loan analysis and result display
- User authentication and profile management
- Loan history tracking

## Tech Stack
Frontend:
- React.js
- HTML, CSS, JavaScript

Backend:
- Django
- REST API

Machine Learning:
- Classification model (trained using loan dataset)
- Model file: loan_model.pkl

Database:
- SQLite

## AI/ML Concepts Used
- Data preprocessing
- Feature selection
- Classification algorithms
- Model evaluation

## How it Works
1. User fills loan application form  
2. Data is processed and sent to backend  
3. ML model predicts risk level  
4. System displays approval/rejection result  

## Project Structure
- Frontend: React components (UI)
- Backend: Django APIs and logic
- ML Model: Trained prediction model
